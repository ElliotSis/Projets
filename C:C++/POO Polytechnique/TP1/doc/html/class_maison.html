void Maison::observerCellulaires ( float  tempsObservation = 60.0f)

Analyse en boucle les cellulaires observés pour savoir si ils sont proches. Si un cellulaire est proche, la maison vérifie si la personne à qui il appartient occupe la maison. Les cellulaires sont observés pour un temps déterminé. Par défaut, la durée est de 60 secondes.

Paramètres
tempsObservationla durée (en secondes) d'observation des cellulaires.
