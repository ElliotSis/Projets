Description détaillée

Classe utilisée dans le cadre du patron de conception MVC.

Le controlleur est le "C" de MVC. Il sert à instancier la vue et le jeu. Il traite les informations reçues par la vue, puis les envoie au jeu et à la vue pour qu'ils se mettent à jour.

bool Controlleur::traiterCaseChoisie ( Case *  caseChoisie)

Traite un coup rentré par l'utilisateur.

Si il y a une pièce sur la case choisie et que celle-ci est jouable, la variable temporaire pièce à déplacer est modifiée. Si la pièce à déplacer est déjà définie et que la case d'arrivée est valide : le coup est joué.

Paramètres
caseChoisieun pointeur sur la case que l'utilisateur a selectionné.
Renvoie
Un booléen valant true si le coup est joué, false sinon.

void Controlleur::traiterChoixMenuPrincipal ( int  choixJeu,
int  choixJ1,
int  choixJ2 
)

Traite le choix du menu principal.

Cette fonction sert à instancier et lancer le jeu une fois que l'utilisateur l'a choisi.

Paramètres
choixUn entier représentant le jeu joué (1 : jeu de dames, 2 : jeu d'othello, 3 : jeu du Puissance 4).
choixJ1Un entier représentant le joueur choisi pour commencer la partie (0 : joueur humain, 1-15 : un joueur artificiel avec une profondeur de recherche choixJ1).
choixJ2Un entier représentant le joueur choisi comme adversaire au joueur 1 (0 : joueur humain, 1-15 : un joueur artificiel avec une profondeur de recherche choixJ2).
